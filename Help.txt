How to run:
- Download File
- Unzip
- Open the "GPRO" folder in your desired IDE (we used Intelleji)
- Open GP/src in the IDE
- Setup JDK (we used version 19)
- Run from Test class
- Follow prompts and explore the code!
- Enter E into command line interface to exit program


UML:
- Once you unzip the file, UML and CRC are located in images folder

JavaDocs:
- Once you unzip the file, javaDocs is located in javaDocs folder
